# DQN-for-CaRL
## Requirements
The requirements are pygame and pytorch:

``pip install pygame``

- https://pytorch.org/get-started/locally/

## Testing the agent
``python play.py``

Hit ``x`` to activate autopilot et ``c`` to deactivate it. 
Hit ``t`` to get a screenshot of the agent's view.
